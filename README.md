# Dizi - Catálogo Dinámico SAAS

Este es el repositorio oficial de **Dizi**, la plataforma definitiva para crear catálogos digitales interactivos.

## Repositorio Oficial

[https://github.com/lovableprojectscx/Dizi.git](https://github.com/lovableprojectscx/Dizi.git)

## Tecnologías

- React (TanStack Start)
- Supabase
- Tailwind CSS
- Lucide React

## Instalación

```bash
npm install
npm run dev
```
